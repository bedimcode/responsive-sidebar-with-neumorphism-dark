# Responsive Neumorphism Sidebar With Dark Mode 

This project is part of a YouTube tutorial in which we build a responsive neumorphism sidebar with dark mode from scratch using HTML CSS & JavaScript

## 🎬 [Watch the Demo & Code](https://youtu.be/jTkimpdbHPk) 

![preview img](/preview.png) 

## 📌 Project Features 
- Fully Responsive Neumorphism Sidebar With Dark Mode  (Mobile First Methodology).
- Clear and semantic HTML structure.
- Use of CSS variables.
- Smooth animations with CSS.
